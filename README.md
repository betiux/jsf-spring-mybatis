jsf-spring-mybatis
==================

Template for building JEE app with JSF, Spring and Mybatis
- Simple form
- Persistence
- Internationalization
- JSF handling exception

Frameworks
- JDK 7
- Tomcat 7 / JBoss 7
- JSF 2.1
- Spring 3.1
- MyBatis 3.2

JSF 2.2 works (out of the box) with Tomcat 7 but not with JBoss 7.1
